Sistema generativo, não acervo fixo. Cada composição usa uma função pseudo-aleatória com seed — o mesmo seed sempre produz o mesmo resultado, mas qualquer seed gera uma variação única. Clique em ↺ Regen Tudo para ver novas variações. Cada card tem seu próprio seed independente para controle granular. A linguagem visual é determinada pela geometria do Alfa, pela paleta ADA e pelo algoritmo de composição — nunca por elementos externos. O resultado é sempre reconhecidamente ADA.
