# Índice de Seções do Manual

Arquivo base: `src/components/sections/`

| # | Arquivo | Conteúdo |
|---|---|---|
| 01 | `LogoSection.astro` | Logo ADA: símbolo, wordmark, variações, mínimos, área de proteção, usos incorretos, downloads SVG/PNG |
| 02 | `PaletteSection.astro` | Paleta de cores: preto, amarelo, branco. Tokens por modo (Padrão, Inversão, Mono). Demos fixas de modo |
| 03 | `TypographySection.astro` | 7 níveis tipográficos. DM Mono, Syne, Cormorant Garamond. Escala, tracking, leading |
| 04 | `VozSection.astro` | Manifesto ANTES. Traços verbais (precisa, ponto de vista, econômica, situada). Vocabulário ADA e evitar |
| 05 | `ElementosSection.astro` | Imagetica: 52 fotos, 11 territórios. Grid, uso, proporções |
| 06 | `MerchandiseSection.astro` | Aplicação em merch: camiseta, bag, adesivo. Regras de cor por material |
| 07 | `MateriaisDigitaisSection.astro` | Email, redes sociais, apresentação. Templates e regras de uso digital |
| 08 | `ComponentesSection.astro` | Componentes UI: .ada-btn, .ada-tag, .ada-field, .ada-card, .ada-nav. Demos interativas |
| 09 | `PromptSection.astro` | Prompt de sistema autônomo para agentes IA operarem como ADA |

## CSS global
`src/styles/tokens.css` — todos os tokens, layout e componentes.

## Layout raiz
`src/layouts/Base.astro` — fontes, SVG symbols inline, scripts globais (dlPng, dlFav, setMode).
