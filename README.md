# ADA Brand Manual

Manual de identidade visual da ADA — Ateliê Digital Analógico.
Site estático interativo com assets para download.

## Stack

HTML / CSS / JS puro · Google Fonts · html2canvas

## Estrutura

```
ada-manual/
├── index.html          ← site completo
├── css/system.css      ← variáveis, layout, componentes
├── js/ada.js           ← menu, scroll spy, taglines
├── js/merch.js         ← seção 10 (merchandise)
└── assets/             ← imagens e wordmark SVG
```

## Rodando localmente

Abrir `index.html` diretamente no browser. Sem build, sem dependências locais.
